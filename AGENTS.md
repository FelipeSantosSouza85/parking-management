# AGENTS.md

## Objective
This repository is a Springboot 3 + Java 21. This system represents a backend service responsible for managing parking garage operations, including vehicle entry and exit processing, parking spot allocation, and revenue calculation.

Primary responsibilities:
This system represents a backend service responsible for managing parking garage operations, including vehicle entry and exit processing, parking spot allocation, and revenue calculation.

The service integrates with an external garage simulator that emits parking events through a webhook interface. These events represent the lifecycle of a vehicle within the parking garage, including entry, parking confirmation, and exit.

The system is responsible for maintaining the internal state of the garage, ensuring spot availability consistency, applying business rules for pricing and occupancy, and calculating parking revenue.

The application exposes a REST API that allows querying the revenue generated by the parking garage based on date and sector.
